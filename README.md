# English with Mom V2

## Đưa lên GitHub Pages
1. Tạo repository mới trên GitHub.
2. Upload toàn bộ file và thư mục trong gói này vào nhánh `main`.
3. Vào Settings → Pages.
4. Chọn Deploy from a branch → `main` → `/root` → Save.
5. Chờ GitHub tạo đường link.

## Cài trên iPhone
Mở link bằng Safari → Chia sẻ → Thêm vào Màn hình chính.

Ảnh MinMin và MoMo chỉ nên dùng trong repository riêng tư hoặc khi gia đình đồng ý công khai.
